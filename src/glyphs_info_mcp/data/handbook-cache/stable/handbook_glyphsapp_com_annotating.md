## [Annotating](#annotation-tool)

The Annotation tool ![](https://handbook.glyphsapp.com/v/content/icons/annotation-tool.svg) (shortcut `A`) adds notes and correction marks to glyph layers.
When the tool is active, the Info box (`View → Show Info`, `Cmd-Shift-I`) turns into a small palette holding a range of annotation tools.

![](https://handbook.glyphsapp.com/v/content/images/root/annotation-subtools.svg)

Choose `Edit → Select All` (`Cmd-A`) to select all annotations in the currently active glyph layer.
Move selected annotations with the arrow keys.
Hold down `Shift` for increments of 10, and `Command` for increments of 100 units.
Press the `Delete` key to remove all selected annotations.

### [Annotation Cursor](#annotation-tool/cursor)

The first tool in the Annotation palette is the Annotation Selection tool ![](https://handbook.glyphsapp.com/v/content/icons/annotation-subtool-selection.svg).
Use it to click-select annotations.
`Shift`-click to select multiple annotations at once.

### [Annotation Text](#annotation-tool/text)

Place snippets of text with the Text Annotation tool ![](https://handbook.glyphsapp.com/v/content/icons/annotation-subtool-text.svg).
Click to add a new text annotation, double-click an annotation to edit its text.
A text snippet can span across multiple lines.
The handle on the right controls the width of the text box.

![](https://handbook.glyphsapp.com/v/content/images/root/annotation-subtool-text.svg)

### [Annotation Arrow](#annotation-tool/arrow)

The Arrow Annotation tool ![](https://handbook.glyphsapp.com/v/content/icons/annotation-subtool-arrow.svg) adds red arrows onto the canvas.
Use the handle on an arrow stem to control its rotation.

![](https://handbook.glyphsapp.com/v/content/images/root/annotation-subtool-arrow.svg)

### [Annotation Circle](#annotation-tool/circle)

The Circle Annotation tool ![](https://handbook.glyphsapp.com/v/content/icons/annotation-subtool-circle.svg) adds red circles onto the canvas.
The handle at the bottom of the circle controls its diameter.

![](https://handbook.glyphsapp.com/v/content/images/root/annotation-subtool-circle.svg)

### [Plus & Minus Annotations](#annotation-tool/plus-and-minus)

Many designers use plus and minus signs to indicate that a counter, a bowl, or a stem should be thickened or thinned.
Click the plus ![](https://handbook.glyphsapp.com/v/content/icons/annotation-subtool-plus.svg) or minus ![](https://handbook.glyphsapp.com/v/content/icons/annotation-subtool-minus.svg) button and then click on the canvas to add the symbols to the relevant area.

![](https://handbook.glyphsapp.com/v/content/images/root/annotation-subtool-plus-minus.svg)
