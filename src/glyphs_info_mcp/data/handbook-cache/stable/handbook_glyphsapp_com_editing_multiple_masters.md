## [Editing Multiple Masters](#editing-multiple-masters)

Font masters are listed in the `Layers` palette.
Click the ![](https://handbook.glyphsapp.com/v/content/icons/invisible.svg)/![](https://handbook.glyphsapp.com/v/content/icons/visible.svg) buttons to show a layer even if it is not currently active.
See [Layers](https://handbook.glyphsapp.com/single-page/#palette/layers "Palette ⯈ Layers") for details.

### [Select All Layers Tool](#select-all-layers-tool)

Use the Select All Layers tool ![](https://handbook.glyphsapp.com/v/content/icons/select-all-tool.svg) (shortcut `Shift-V`) to edit the paths of all visible layers at the same time.
Click the ![](https://handbook.glyphsapp.com/v/content/icons/invisible.svg)/![](https://handbook.glyphsapp.com/v/content/icons/visible.svg) buttons in the `Layers` palette to show or hide layers.
Switch between the Select and Select All Layers tool by clicking and holding the Select ![](https://handbook.glyphsapp.com/v/content/icons/select-tool.svg) icon, then choose the desired tool from the menu:

![](https://handbook.glyphsapp.com/v/content/images/root/tools-menu-select.png)

### [Show All Masters](#show-all-masters)

When in Edit View, choose `Edit → Show All Masters` to add all master layers of the current glyph to the Edit View tab.
This command also inserts Intermediate and Alternate layers.

### [Keep Layer Selection in Sync](#keep-layer-selection-in-sync)

Select the `Edit → Keep Layer Selection in Sync` option to keep the current selection when changing masters.
This option only works when working on a compatible glyph.
