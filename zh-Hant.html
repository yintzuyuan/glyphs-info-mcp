安裝方式

步驟一：安裝 uv 套件管理器 
curl -LsSf https://astral.sh/uv/install.sh | sh
步驟二：設定 Claude Desktop 
// ~/Library/Application Support/Claude/claude_desktop_config.json
{
  "mcpServers": {
    "glyphs-info": {
      "command": "uvx",
      "args": ["glyphs-info-mcp"]
    }
  }
}

重新啟動 Claude Desktop 即可開始使用！
